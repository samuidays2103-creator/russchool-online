## ГЛОБАЛЬНЫЕ ПРОВЕРКИ (на каждом скриншоте)

### Пустые области
- [ ] Нет пустых белых блоков больше 100px высотой
- [ ] Нет серых полос/линий без контента
- [ ] Нет оранжевых скобок/артефактов (border-left на пустых элементах)
- [ ] Футер не "плавает" посреди страницы — прилегает к контенту

### Текст
- [ ] Нигде нет слова "курс/курсы" — только "предмет/предметы" или "урок/уроки"
- [ ] Нет английского текста (кроме технических страниц admin)
- [ ] Нет технических названий (LIT-1, MAT-1, RUS-1, OKR-1) — только полные названия
- [ ] Нет Moodle-терминов ("Элемент курса", "Активность", "Экземпляр")

### Навбар
- [ ] Только: Логотип | Мои предметы | Расписание | Оценки | Сообщения
- [ ] Нет "В начало", "Дополнительно", "Личный кабинет" (дубли)
- [ ] Нет синих рамок/focus rings при клике
- [ ] Логотип EasyDays виден (белый на тёмно-синем)

### Секции курса
- [ ] Каждая секция имеет описание (не пустая)
- [ ] Описание содержит: что изучаем, план урока, домашнее задание
- [ ] BBB карточка "Онлайн-урок" видна и стилизована (синий gradient)
- [ ] Нет пустых activity-header блоков (серые полосы)

### Карточки предметов (dashboard)
- [ ] Нет фильтров/сортировки/поиска (скрыты)
- [ ] Нет shortname (LIT-1)
- [ ] Единый gradient фон на всех карточках
- [ ] Прогресс-бар виден

### Для ребёнка 6-10 лет
- [ ] Всё понятно без объяснений
- [ ] Нет лишних кнопок ("Новое событие", "Легенда событий")
- [ ] При нажатии на любую секцию — есть контент, не пустота
