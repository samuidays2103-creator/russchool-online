## 1. SKYSMART / SKYENG

### Общее
Skysmart — детское крыло Skyeng (английский). Единая дизайн-система StratosSkyeng охватывает оба продукта. Один из самых технологически продвинутых дизайнов в российском EdTech.

### Цветовая палитра
- **Основной голубой**: `#00aefa` (кнопки, активные элементы, ссылки)
- **Голубой hover**: `#1fbbff`
- **Оранжевый (детский)**: `#ff6e01`, `#ffb136`, `#fb8023`
- **Фон оранжевого градиента**: `radial-gradient(ellipse 95% 83% at bottom right, #ffb136, transparent)`
- **Жёлтый**: `#fad800`, `#ffc700`
- **Тёмный текст**: `#04121b` (почти чёрный, синеватый)
- **Нейтральный фон**: `#f4f5f6`, `#f4f4f4`
- **Белый**: `#ffffff`
- **Загрузочный экран**: `#f1f5fb` (светло-голубой)
- **Детские акценты**: розовый `#ffebf5`, персиковый `#fff2e0`
- **Фон навигации кабинета**: `#5ab2e4`

### Типографика
- **Фирменный шрифт**: StratosSkyeng (собственная гарнитура)
- Начертания: Light (300), Regular (400), Medium (500), SemiBold (600), Bold (700)
- Fallback: Helvetica Neue, Arial, sans-serif
- H1: 36–64px (мобиль → десктоп), font-weight 500
- H2–H3: 20–32px, font-weight 500
- Основной текст: 16–18px, line-height 1.55
- letter-spacing: от -0.5px до -0.1px (отрицательный для компактности)

### Структура главной страницы (skysmart.ru)
**Hero-секция:**
- Полноэкранная с градиентным оранжевым фоном (bottom-right)
- Высота: 646px (десктоп) → 1100px (мобиль — больше места для формы)
- Крупный заголовок белого цвета
- Форма захвата лида: телефон + кнопка
- Иллюстрация девушки/ребёнка абсолютно позиционирована
- Декоративные стрелки-подсказки (hint arrows) с анимацией fadeIn (2 сек задержка)

**Навигация:**
- Sticky header: 56px (мобиль) / 64px (десктоп)
- Три темы: light / dark / transparent (меняется при скролле)
- На мобилях до 1024px — упрощённая/скрытая

**Breakpoints**: 320px, 375px, 430px, 480px, 768px, 1024px, 1366px, 1920px

### Интерфейс ученика (edu.skysmart.ru)
- URL входа: `https://edu.skysmart.ru/login-student`
- Дашборд: результаты по предметам, невыполненные задания, материалы для самостоятельного обучения, олимпиады
- Цвет кабинета: голубой `#039be5` для ссылок, `#5ab2e4` для навигации
- Режим заданий: статусы "Надо решить" / "Решённые"
- Таймер выполнения заданий
- Интерактивная тетрадь с автоматической проверкой
- Кнопки: "Создать задание", "Сдать учителю"
- border-radius кнопок: 8px

### Интерфейс учителя
- Создание/удаление заданий по предметам
- Установка сроков сдачи
- Электронный журнал с прогрессом
- Чат с учеником + доска для рисования/черчения
- Создание тестов и викторин прямо в интерфейсе

### Иллюстрации и стиль
- 2D emoji как иллюстрации и стикеры (фирменная черта Skysmart, не Skyeng)
- WebP формат с PNG fallback
- Яркие, дружелюбные к детям
- Стрелки и декоративные элементы как часть дизайна

### Мобильная адаптация
- Полностью адаптивный с 8 брейкпоинтами
- Safe-area-inset для устройств с выемками (iPhone)
- Форма лида прилеплена к нижней части экрана на мобилях (sticky CTA)
- Мобиль — первичная платформа

### Сильные стороны UI
- Фирменный шрифт создаёт уникальную идентичность
- Оранжевый + голубой — тёплая/доверительная комбинация для детей
- Sticky CTA на мобиле — высокая конверсия
- 2D emoji как иконки — современно, узнаваемо
- Backdrop-filter blur на хедере — стеклянный эффект

### Слабые стороны
- Собственный шрифт недоступен сторонним разработчикам
- Оранжевый + голубой — очень "корпоративная" комбинация, много конкурентов используют похожее
- Загрузочный экран перед кабинетом — React SPA, медленный старт
- Сложность дизайн-системы — воспроизвести сложно

## 2. FOXFORD

### Общее
Один из старейших российских EdTech-игроков (с 2012). Фокус на школьниках 5–11 классов, олимпиадная подготовка, ЕГЭ. Имеет публичный UI Kit.

### Цветовая палитра (из UI kit)
- **Связующий цвет менеджера версий**: `#f5f5f6` (нейтральный фон)
- **Синий для ссылок**: `#1673b1`
- **Оранжевый при hover**: `#ee9900`
- Точная палитра продукта в публичном доступе ограничена, но из обзоров: тёмно-синий как основной, оранжевый/жёлтый как акцент

### Структура
- Основной сайт недоступен без авторизации (401) — закрытая платформа
- Имеет полноценный UI Kit: `ui.foxford.ru` с версионированием (latest, beta, v1.x)
- Это признак зрелой дизайн-системы

### Личный кабинет ученика
- Программы обучения (курсы, репетиторы, мини-группы)
- План занятий
- Календарь уроков
- Система баллов опыта за задачи (геймификация)
- Отслеживание успеваемости
- Текущие задания
- Информация о ближайших событиях

### Интерфейс учителя
- Дашборд с программами
- Расписание
- Прогресс учеников

### Сильные стороны
- Зрелая дизайн-система с UI Kit
- Геймификация баллов опыта
- Глубокая функциональность для серьёзного обучения
- Репутация академической платформы

### Слабые стороны
- Закрытость (нельзя посмотреть без регистрации) — потеря потенциальных клиентов
- Более серьёзный/корпоративный тон — может отпугивать младших школьников
- Фокус на старшеклассников, не дети 6–10 лет

## 3. UCHI.RU

### Общее
Крупнейшая российская образовательная платформа по охвату: 18+ млн учеников (2024). Фокус на 1–4 классы (есть планы расширения). Полностью геймифицирована.

### Интерфейс (из доступных описаний)
- Страница грузится как SPA с анимированным спиннером (JS-driven)
- Основной экран: flexbox-центрированный загрузчик

### Геймификация (фирменная черта)
- Персонажи "Завры" (ящерицы/динозаврики) — маскоты платформы
- За решение задач выдаются:
  - Награды и дипломы
  - Звания
  - Баллы
- Марафоны — соревнования между классами и школами
- Адаптивная система: если тема сложная — предлагает подводящие задания

### Цветовая схема
- Недоступна напрямую (SPA с JavaScript), но из описаний:
- Яркая, дружелюбная к детям
- Тёплые цвета, характерные для детских платформ

### Структура личного кабинета ученика
- Задания от учителя
- Самостоятельная тренировка
- Марафоны
- Достижения и награды
- Рейтинг класса

### Сильные стороны
- Лучшая геймификация в российском EdTech для начальной школы
- Персонажи создают эмоциональную связь
- Адаптивная система обучения
- Гигантская база контента (73 000+ заданий по некоторым оценкам)
- Сильная вирусность через марафоны

### Слабые стороны
- Ориентация преимущественно на 1–4 классы
- Перегруженность геймификацией — может отвлекать от учёбы
- Зависимость от JavaScript — медленный первый экран
- Нет живых уроков (только самостоятельная работа)

## 4. СБЕРКЛАСС (sberclass.ru)

### Общее
Корпоративный продукт Сбера для дополнительного образования. Позиционируется как платформа с "образовательными траекториями" и AI-помощником СберКот.

### Цветовая палитра
- Белый основной фон
- Чёрный текст для заголовков
- Синий для ссылок и кнопок (корпоративный синий Сбера)
- Зелёный (фирменный Сбера) в логотипе
- Минималистичная палитра, корпоративный стиль

### Структура главной страницы
**Навигация:** логотип слева + "Войти" справа (минимальная)

**Hero:** "СБЕР КЛАСС — цифровая платформа для дополнительного образования детей" + "Узнать больше"

**Блоки:**
1. "Раскройте потенциал ребёнка" — 5 иконочных карточек:
   - Образовательная траектория
   - Развитие навыков
   - Достижение целей
   - Отслеживание прогресса
   - Поддержка мотивации

2. Направления: 73 000+ уникальных заданий, финансовая/цифровая/экологическая грамотность, курс "Учусь учиться"

3. Новости — 3 карточки с изображениями

4. СберКот — AI-диалоговый помощник

**Footer:** контакты, документы, часы поддержки

### Иллюстрации
- Современные минималистичные PNG @2x
- Без ярких детских персонажей
- Корпоративный стиль

### CTA
- "Войти", "Узнать больше", "Подписаться", "Отправить заявку"

### Сильные стороны
- Доверие бренда Сбера
- AI-ассистент (СберКот)
- Большая база заданий
- Чистый минималистичный дизайн

### Слабые стороны
- Слишком корпоративный — не детский и не тёплый
- Минимальная навигация скрывает функциональность
- Закрытость за авторизацией
- Ощущение "проекта госкорпорации", а не образовательного пространства

## 5. ДНЕВНИК.РУ

### Общее
Образовательная платформа для школ (дневник + журнал + социальная сеть). Интегрируется с Госуслугами. Фокус на B2B/B2G (школы), не B2C.

### Структура главной страницы
**Навигация:** "О компании" + "Поддержка" + "Войти" + "Подключить ОО"

**Hero:** "Цифровая образовательная платформа, которая делает образование в России качественным и доступным"

**Блоки по аудиториям:**
- Преподавателям — автоматизация
- Родителям — близость к ребёнку
- Учащимся — всё для учёбы и внеклассной работы
- Госорганам — онлайн-мониторинг

**Иллюстрации:** teacher.png, parents.png, childrens.png, officials.png — реалистичные фото людей

**Footer:** Telegram, ВКонтакте, YouTube; возрастная маркировка 6+

### Личный кабинет ученика
Рабочий стол: календарь, Лента событий, Сообщения, Новости
Навигация: Моя страница (был "Мой дневник"), Образование (была "Школа"), Приложения (был "Центр приложений")
Иконки: лупа (поиск), сообщения, приглашения, запросы дружбы — с счётчиком уведомлений
Функции: личные сообщения, портфолио с сертификатами, электронный журнал

### Сильные стороны
- Интеграция с Госуслугами (авторизация)
- Официальный статус — доверие родителей и школ
- Многофункциональность (дневник + журнал + соцсеть)
- Охват через государственные школы

### Слабые стороны
- Дизайн устаревший — социальная сеть образца 2010-х
- Не адаптирован под мобильные как нативное приложение
- Ориентирован на школы, а не на онлайн-обучение
- Нет инструментов для живых уроков

## 6. ЯНДЕКС.ОБРАЗОВАНИЕ (education.yandex.ru)

### Общее
Зонтичная платформа Яндекса для всего образования: от школьников до корпоративного. Включает Яндекс Учебник (1–9 класс), Яндекс Лицей (программирование), ШАД, Практикум.

### Цветовая схема
- Акцентный цвет: `#DCD26E` (мягкий жёлто-зелёный, необычный выбор)
- Светлый фон (hrLight theme)
- Тёмный текст
- Нейтральная серо-чёрная палитра
- Шрифты: предзагружаемые woff2 (12 файлов) — корпоративная гарнитура Яндекса

### Структура главной
**Hero:** "Объединяем людей, технологии и образование"

**Навигация:** Школьникам / Студентам / Абитуриентам / Партнёрам / Хендбуки / Журнал / AI

**Карточки проектов (сетка 5):**
- Яндекс Учебник — интерактивные задания 1–9 классы
- Яндекс Лицей — базовое IT-образование
- ШАД — ML и big data
- Программы в университетах
- Яндекс Практикум — онлайн-переквалификация

**Дополнительно:** Журнал, Карты IT-навыков, Хендбуки, Логотипы вузов-партнёров (карусель)

### Типографика
- heading-m для основных заголовков
- data-weight="regular", data-color="primary" (CSS Custom Properties)
- 12 предзагружаемых woff2 файлов

### Сильные стороны
- Экосистема Яндекса = доверие и узнаваемость
- Сильная навигация по аудиториям
- AI-раздел — актуальность
- Отдельные продукты с проработанными UX

### Слабые стороны
- Акцентный цвет `#DCD26E` неочевиден и слабоконтрастен
- Слишком широкий охват — не фокусируется на начальном/среднем школьном возрасте
- B2B/партнёрская ориентация на главной — не для детей и родителей

## 7. SKYENG (skyeng.ru) — взрослый продукт для сравнения

### Цветовая палитра
- **Основной голубой**: `#00aefa` (= Skysmart, единая система)
- **Hover**: `#1fbbff`
- **Оранжевый**: `#ff6d00`, `#ff9e00`
- **Жёлтый**: `#fad800`, `#ffc700`
- **Тёмный**: `#04121b`
- **Нейтральный фон**: `#f4f5f6`

### Структура главной (skyeng.ru)
**Hero:** полноэкранная с адаптивными фонами (5 breakpoints: 375/768/1024/1366/1920px)
Плавающие карточки со статистикой (lessons / students / teachers)
Заголовок: 26px (мобиль) → 52px (десктоп), font-weight 500, белый, поворот -5° на мобилях

**Navbar:**
- Sticky, 56px (мобиль) / 64px (десктоп)
- Эффект: backdrop-filter: blur(16px) — стеклянный на десктопе
- Скрывается при скролле вниз, появляется при скролле вверх

**CTA-кнопка:**
- Цвет: `#00aefa`
- Форма с полями: имя, email, телефон
- Позиция: абсолютная, внизу контейнера

### Интерфейс урока Skyeng
- Собственная платформа Vimbox (не Zoom/BigBlueButton)
- Интерактивный учебник (картинки, видео, правила, упражнения)
- Drag-and-drop задания
- Подсветка результата: зелёный (правильно) / красный (ошибка)
- Автодобавление сложных слов в словарь
- ИИ-генерация контента и обложек (2025)

### Личный кабинет
Детский кабинет отличается от взрослого (другая навигация, больше предметов)
Разделы взрослого кабинета:
- Домашка (задания и тесты)
- Обучение (активности, практика, уроки)
- Практика (словарь, грамматика, навыки, видеопрактика, симуляции)
- История занятий + заметки
- Расписание (виджет на главной)
- Чат с поддержкой и преподавателем
- Оплата, промокоды, история платежей
- Реферальная программа ("Подарок")
- Семья и курсы

### Сильные стороны
- Лучший UX взаимодействия учитель-ученик
- Стеклянный эффект хедера — современно
- ИИ-персонализация контента — тренд 2025
- Отдельный детский кабинет — правильное решение
- Мобильное приложение (iOS 4.8, Android 4.5)

### Слабые стороны
- Проприетарная платформа Vimbox — не нужна для нашей школы
- Дорогостоящая инфраструктура
- Ориентирован на взрослых прежде всего

## 8. KHAN ACADEMY (Wonder Blocks Design System)

### Принципы дизайна Wonder Blocks
1. **Empowering** — студент чувствует силу и свободу воли, не патернализм
2. **Credible** — чистый, структурированный дизайн = доверие
3. **Flexible** — адаптируется к разным аудиториям
4. **Joyful** — процесс учёбы полон борьбы, важно отмечать маленькие победы

### Цветовая система
Была: 58+ цветов (хаос)
Стала: 18 цветов с функциональными назначениями
- Синий = действия
- Красный/зелёный = предупреждения/успех (только с текстом)
- Отказ от предметных цветов → многоцветные иллюстрации

### Типографика
Было: 8+ гарнитур, 119+ стилей
Стало: 5 гарнитур, 14 стилей
Информационная плотность: +11–26% без ущерба читаемости

### Компоненты
Кнопки: было 50+ разновидностей → чёткий организованный набор
Поповеры, таблицы, модальные окна

### Структура (из описаний)
- Студент видит курсы в виде "юнитов" с прогресс-барами
- Каждый урок — видео + упражнения
- Система энергетических очков (XP)
- Ачивки и значки

### Сильные стороны
- Зрелейшая дизайн-система из всех рассмотренных
- Принцип Joyful — отмечать маленькие победы: важнейший UX-принцип для детей
- Экономия: 18 цветов вместо 58 — чистота
- Доступность контента (бесплатно)
- Мультиязычность

### Слабые стороны
- Дизайн воспринимается как "бесплатный" (что и есть)
- Визуально скромнее коммерческих конкурентов
- Не адаптирован к российской школьной программе

## 9. DUOLINGO — мировой лидер геймификации

### Цветовая палитра
- **Лаймовый зелёный**: `#58CC02` (основной бренд)
- **Маска-зелёный**: `#89E219` (вторичный)
- Оранжевый и жёлтый (акценты клюва совы)
- Белый фон основного контента
- Тёмно-серый текст

### Геймификация (исчерпывающий список)
- **Streaks (Серии)** — количество дней подряд занятий; главный крючок удержания
- **Hearts (Сердца/Жизни)** — 5 попыток на урок, теряются при ошибках; создаёт срочность
- **XP (Очки опыта)** — за каждое задание
- **Leaderboards (Таблицы лидеров)** — лиги, глобальный рейтинг; социальная конкуренция
- **Levels (Уровни)** — система прогрессии
- **Progress bars (Прогресс-бары)** — в каждом уроке
- **Streak freeze** — можно "купить" защиту серии
- Персонаж Дуо (сова) — активный участник, поздравляет, напоминает

### Мобильный интерфейс
- Полностью иллюстративный — всё нарисовано
- Яркие цвета throughout
- Белый фон + яркие модули
- Минимализм в навигации
- Простота как главный принцип

### Структура урока
- Введение (несколько слайдов)
- Задания (4-7 типов чередуются)
- Celebration screen при завершении
- XP recap

### Сильные стороны
- Лучшая в мире система удержания через streaks
- Персонаж Дуо создаёт личную связь
- Мобиль-первый дизайн идеален
- Короткие уроки (3-5 мин) = легко начать
- Бесплатный базовый доступ = вирусность

### Слабые стороны
- Геймификация сильнее образовательного содержания (критика учёных)
- Не подходит для серьёзного академического обучения
- Сердца создают стресс и могут демотивировать новичков

## 10. SKYENG VIMBOX vs BIGBLUEBUTTON — сравнение платформ видеоуроков

Vimbox (Skyeng):
- Проприетарная разработка
- Интегрированный учебник
- Drag-and-drop задания прямо в видеоуроке
- Автословарь

BigBlueButton (наша школа):
- Open-source, интегрируется с Moodle
- Общая доска, чат, разбивка на группы
- Запись уроков
- Менее интегрирован с учебными материалами
- Требует отдельного контента

**Вывод:** BBB достаточен для живых уроков малыми группами, но нужен хороший интерфейс вокруг него в Moodle.
