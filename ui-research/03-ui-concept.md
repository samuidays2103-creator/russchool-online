**Версия:** 1.1 | **Дата:** Март 2026
**Аудитория:** дети 6–16 лет, родители, учителя | **Рынок:** Россия (80%) + диаспора ЮВА/ОАЭ/Германия (20%)

## 1. ФИЛОСОФИЯ ДИЗАЙНА

### Три кита
1. **Тепло** — как у хорошего учителя. Не корпоративная холодность СберКласса. Не игровая суета Учи.ру. Уютное образовательное пространство, в которое хочется возвращаться.
2. **Доверие** — как у серьёзной школы. Не кричащие скидки и агрессивные CTA. Не перегруженный интерфейс. Чистота, структура, академичность.
3. **Современность** — как у лучших продуктов 2024–2025. Адаптивность, скорость, иллюстрации, микроанимации.

### Принципы (по Khan Academy Wonder Blocks)
- **Empowering:** ребёнок чувствует, что он способен, а не что его тестируют
- **Credible:** чистый, структурированный дизайн = доверие родителей
- **Joyful:** маленькие победы отмечаются ярко — выполнил задание, пришёл на урок
- **Warm:** в отличие от Khan Academy, добавляем эмоциональное тепло (важно для диаспоры, которая скучает по дому)

## 2. ЦВЕТОВАЯ ПАЛИТРА

### Основные цвета

```
Primary (Первичный) — глубокий сине-фиолетовый:
  --color-primary:      #4F46E5   (indigo-600)
  --color-primary-dark: #3730A3   (indigo-800, для hover/active)
  --color-primary-light:#EEF2FF   (indigo-50, светлый фон)

Accent (Акцент) — тёплый янтарный:
  --color-accent:       #F59E0B   (amber-500)
  --color-accent-dark:  #D97706   (amber-600, для hover)
  --color-accent-light: #FEF3C7   (amber-100, светлый фон)

Success (Успех) — мятный зелёный:
  --color-success:      #10B981   (emerald-500)
  --color-success-light:#D1FAE5   (emerald-100)

Warning (Предупреждение):
  --color-warning:      #F97316   (orange-500)
  --color-warning-light:#FFEDD5   (orange-100)

Error (Ошибка):
  --color-error:        #EF4444   (red-500)
  --color-error-light:  #FEE2E2   (red-100)
```

### Нейтральные (основа интерфейса)

```
  --color-bg:           #FAFAF9   (stone-50, тёплый белый — не холодный #fff)
  --color-bg-card:      #FFFFFF   (белый для карточек)
  --color-bg-section:   #F5F0EB   (тёплый бежевый для секций — отличает нас)
  --color-border:       #E7E5E4   (stone-200)
  --color-border-focus: #4F46E5   (primary при фокусе)

Текст:
  --color-text-primary: #1C1917   (stone-900 — почти чёрный, тёплый)
  --color-text-second:  #57534E   (stone-600)
  --color-text-muted:   #A8A29E   (stone-400)
  --color-text-inverse: #FFFFFF
```

### Предметные цвета (для иконок карточек)

```
  --color-subject-russian:  #8B5CF6   (фиолетовый — буква Я)
  --color-subject-math:     #3B82F6   (синий — цифры)
  --color-subject-reading:  #EC4899   (розовый — книга)
```

### Почему такая палитра?

- **Индиго (#4F46E5)** — не типичный "детский" голубой Skysmart/Skyeng. Умный, немного "волшебный", ассоциируется с качественным образованием. Отличает нас от всех конкурентов.
- **Янтарный (#F59E0B)** — тёплый акцент. Не кислотно-оранжевый как у Skysmart, а благородный, золотой. Создаёт ощущение ценности.
- **Stone фоны (#FAFAF9, #F5F0EB)** — тёплые, уютные. Не холодный белый как у СберКласса. Ощущение "домашнего" пространства.

## 3. ТИПОГРАФИКА

### Шрифты

```css
/* Основной шрифт — Golos Text (российская разработка, отличная кириллица) */
@import url('https://fonts.googleapis.com/css2?family=Golos+Text:wght@400;500;600;700&display=swap');

/* Акцентный/декоративный (для заголовков hero и спецэлементов) */
@import url('https://fonts.googleapis.com/css2?family=Nunito:wght@600;700;800&display=swap');

--font-main:    'Golos Text', -apple-system, system-ui, sans-serif;
--font-display: 'Nunito', 'Golos Text', sans-serif;
```

**Почему Golos Text:** создан для государственных и социальных сервисов в России. Кириллица — главный приоритет. Читабельность на экране идеальна. Выглядит профессионально и дружелюбно одновременно. Бесплатен.

**Почему Nunito для заголовков:** округлые терминалы создают ощущение доброты и дружелюбия — важно для детской аудитории. Хорошо работает для коротких заголовков.

### Размерная шкала

```css
/* Заголовки — Nunito */
--text-h1:      clamp(32px, 5vw, 56px);   font-weight: 800
--text-h2:      clamp(24px, 4vw, 40px);   font-weight: 700
--text-h3:      clamp(20px, 3vw, 28px);   font-weight: 700
--text-h4:      clamp(18px, 2.5vw, 22px); font-weight: 600

/* Текст — Golos Text */
--text-xl:      20px;  line-height: 1.6
--text-lg:      18px;  line-height: 1.6
--text-base:    16px;  line-height: 1.65
--text-sm:      14px;  line-height: 1.5
--text-xs:      12px;  line-height: 1.4

/* letter-spacing */
Заголовки: -0.3px ... -0.5px (плотнее)
Текст:      0px (обычный)
Кнопки:    +0.2px (чуть шире для читаемости)
```

## 9. МОБИЛЬНЫЕ ПАТТЕРНЫ

### Критические решения для мобиля (375px, 70% трафика)

**1. Bottom Navigation Bar (вместо бургера)**
```
┌─────────────────────────────────────────────────────────────┐
│  [Главная]  [Уроки]  [+]  [Задания]  [Профиль]             │
│     🏠         📚    большая   📝        👤                  │
│                    кнопка                                   │
└─────────────────────────────────────────────────────────────┘
```
Большая центральная кнопка "+" — войти в урок или сдать задание.

**2. Sticky CTA на главной странице**
```
┌─────────────────────────────────────────────────────────────┐
│  [Записать ребёнка на бесплатный урок]                      │
│  ──────────────────────────────────── fixed bottom 16px    │
└─────────────────────────────────────────────────────────────┘
```

**3. Размер касания (touch targets)**
- Минимум 44×44px для всех интерактивных элементов
- Отступы между кликабельными элементами: 8px

**4. Карточки курсов — полная ширина**
На мобиле: одна карточка = полная ширина, горизонтальный скролл для нескольких

**5. Расписание — горизонтальный скролл**
Таблица расписания на мобиле: sticky первая колонка (время), горизонтальный скролл вправо

**6. Формы — один вопрос за раз**
Не длинная анкета. Шаг 1: имя ребёнка. Шаг 2: возраст. Шаг 3: телефон. Прогресс-бар.

## 10. АНИМАЦИИ И МИКРОВЗАИМОДЕЙСТВИЯ

### Принципы
- Все анимации: 150–300ms
- Easing: ease-out для входа, ease-in для выхода
- Уважать prefers-reduced-motion

### Конкретные анимации

```css
/* Кнопки */
.btn { transition: background 0.2s ease, transform 0.1s ease, box-shadow 0.2s ease; }
.btn:hover { transform: translateY(-1px); }
.btn:active { transform: translateY(0px); scale: 0.98; }

/* Карточки */
.card { transition: box-shadow 0.2s ease, transform 0.2s ease; }
.card:hover { transform: translateY(-4px); }

/* Прогресс-бары (при загрузке страницы) */
.progress-bar {
  width: 0;
  animation: fill-progress 0.8s ease-out 0.3s forwards;
}

/* Появление элементов при скролле */
.fade-in-up {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.4s ease, transform 0.4s ease;
}
.fade-in-up.visible { opacity: 1; transform: translateY(0); }

/* Achievements (при получении) */
@keyframes achievement-pop {
  0%   { transform: scale(0) rotate(-10deg); opacity: 0; }
  70%  { transform: scale(1.2) rotate(3deg); opacity: 1; }
  100% { transform: scale(1) rotate(0deg); opacity: 1; }
}

/* Streak fire (при последовательных посещениях) */
@keyframes fire-pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}
```

### Hover-состояния навигации
```css
nav a {
  position: relative;

nav a::after {
```

## 11. СТИЛЬ ИЛЛЮСТРАЦИЙ

### Концепция: "Warm Flat 2D"

**Характеристики:**
- Flat (плоские), без реализма
- Тёплые, слегка приглушённые тона (не кислотные)
- Детали без перегрузки
- Разнообразие персонажей (разные национальности, т.к. диаспора)
- Ситуации: обучение, общение, семья

**Что изображаем:**
- Hero: ребёнок сидит за столом, перед ним светящийся экран с учителем, за окном — тропический/зарубежный пейзаж (или снежный, для зон СВО)
- Иконки предметов: книга с буквой "Я" (русский), счёты или формула (математика)
- Аватары учителей: реальные фото (важно для доверия)
- Достижения: звёздочки, медали, сертификаты в flat-стиле

**Рекомендуемые источники иллюстраций:**
- Undraw.co (SVG, кастомизируемые цвета)
- Humaaans (персонажи)
- Storyset (тематические сцены)
